# React + Vite

## Install Dependancy and run project

1. Do npm install.
2. Do npm run dev
3. visit http://localhost:5174/

## Test Application using cypress

1. Install cypress  npm install cypress --save-dev
2. Run commond npx cypress open
3. Choose E2E Testing options
4. Choose any browser.
3. Click on spec.cy.js file.
4. See the result.
